Place your product images in this folder if you want to serve them locally.

For example, you can add files like:
- cob-light.jpg
- smd-downlight.jpg
- pk-moon-light.jpg
- energy-saver.jpg
- ice-panel-moon-light.jpg

Then in code you can reference them as:
- /products/cob-light.jpg
- /products/smd-downlight.jpg
- /products/pk-moon-light.jpg
- /products/energy-saver.jpg
- /products/ice-panel-moon-light.jpg
